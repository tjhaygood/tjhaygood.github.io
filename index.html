Discord Bots

These applications are user-facing chat bots that serve users' requests. These requests include a variety of chat games, quick ways to retrieve and display information, playing sounds in voice channels, and Discord server management. Both of the bots that I manage are written in Python.

One bot is a general-purpose bot that plays sounds in voice channels, plays chat games, and uses web scraping to retrieve information from various websites. This bot also collects server statistics upon request and manages a user ELO system for Super Smash Brothers, using a common ELO algorithm and a MongoDB database to store users' ELO scores.

The other bot is a player statistic tracking bot for the popular first-person shooter Rainbow Six: Siege. Using HTTP requests, the bot refreshes its cache of player rankings and in-game statistics periodically and upon user request to track and notify users of when their in-game rank changes and display those rank changes. The bot also manages players' roles on a Discord server based upon their rank changes.

Fellow TV Addicts

Fellow TV Addicts is a cloud-based web application where users can watch YouTube content together. Users can create and enter "rooms" where a single YouTube stream plays, and each user's positions in the stream are synced together in real time. When one user pauses the stream, moves to another position in the stream, or changes which video is being shown, the changes to the stream are reflected to every other member of the room.

The application also features a profile and friend system, where users can express themselves and maintain connections to other users. Beyond the in-room chat, users can also send direct messages to each other, and see when their friends are online, or using the platform.

This application was built on the MERN stack: MongoDB, Express.js, React.js, Redux.js, and Node.js.

Pen Pals

Pen Pals is a social networking platform for book readers to find each other, based on the books thy read. Users must register for an account on the platform, and upon registering can submit a profile picture, a bio, and can enter their favorite books, book authors, and book genres. After creating an account, users can search for other users with similar interests and message them.

This application was built using Flask (a micro web framework written in Python), HTML, and MySQL.

LED GUI

This application, written in Java, is a GUI used to control WS2812B singly-addressable light strips using serial communication with an Arduino Uno. The GUI consists of 8 custom profiles that load color sets and play animations on the light strip. The GUI can also be used to control which lights are turned on, the hue of all lights, and the brightness of all lights.
